# ArguMentor

An AI writing assistant for Hong Kong Grade-11 students that goes beyond grammar checks —
designed around **Toulmin's six components of argument** to coach claims, evidence,
warrants, and rebuttals.

**Live demo:** https://piggymad.github.io/argumentor/

## What it does

- **Workspace** — write argumentative essays with a live Toulmin-component detector and grammar / style feedback
- **AI Tutor** — Socratic dialog that asks rather than answers, scaffolding the six Toulmin components
- **Teacher Review** — hybrid AI + human marking; teacher's note is what the student sees first
- **Progress Dashboard** — radar over the six Toulmin dimensions + score-over-drafts trend
- **Resources** — worked examples grounded in HK ESL contexts

## Theoretical grounding

- Toulmin's Model of Argumentation (Turós et al., 2025)
- Sociocultural / Vygotskian "more knowledgeable other"
- Process Writing Theory — revision is tracked, not erased

## Stack

Plain HTML / CSS / vanilla JavaScript — fully static, no backend. Deployable to any static host.

## Research

This prototype was developed and evaluated as part of an MSc TDLL capstone project (HKU, 2026).
A randomised controlled trial with 74 HK Grade-11 students showed a large effect on argumentative
writing quality (ANCOVA F(1,71) = 173.79, p < .001, η²p = .71), with the largest gain on
warrant + rebuttal (+1.57 / 7 points).
